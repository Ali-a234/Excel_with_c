نام پروژه : excel with c
این پروژه یک spreadsheet ساده مانند قابلیت های پایه ای Excel است که با استفاده از زبان C پیاده سازی شده 


 قابلیت ها :

--  تعیین مقدار برای سلول ها با ادرس هایی مانند A1 B2

-- محاسبه ی فرمول های ریاضی با استفاده از ارجاع به سلول ها

-- پشتیبانی از عملیات ریاضی پایه تا پشتیبانی از توابع استاندارد

-- ذخیره ی اطلاعات spreadsheet در فایل

-- بارگذاری اطلاعات از فایل

-- مدیریت خطا بدون توقف یا کرش کردن



ساختار های پروژه :

-- main.c
مدیریت منوی برنامه و ارتباط بین بخش ها 

-- sheet.h\sheet.c
مدیریت سلول ها و نگهداری داده ها در حافظه 

-- formula.c\formula.h
محاسبه و پردازش فرمول های ریاضی 

-- error.h\error.c
مدیریت و ارجاع و نمایش پیام های خطا

-- file.c\file.h
ذخیره و بازیابی اطلاعات spreadsheet از فایل



این پروژه بصورت ماژولار طراحی شده تا امکان توسعه ی قابلیت های بیشتر
مانند فرمت های فایل پیشرفته تر یا توابع بیشتر فراهم باشد 



اعضای گروه و مسئولیت ها:


نفر اول (آدریانا معصومی) :طراحی و مدیریت main و sheet

نفر دوم (امیرحسین بربری) :طراحی و پیاده سازی فرمول های ریاضیاتی

نفر سوم (دیانا وزیری) :مدیریت خطا ها 

نفر چهارم (علی آقایی) :پیاده سازی بخش مربوط به ذخیره و بارگذاری فایل ها
